AniCat Android Application Downloads

This folder contains APK files for the AniCat Android application.

Current files:
- anicat-android.apk - Main application APK file (Demo version)

Installation instructions:
1. Download the APK file
2. Enable "Unknown sources" in your Android settings
3. Install the downloaded APK file
4. Launch AniCat and enjoy!

System requirements:
- Android 7.0 and above
- 100MB free storage space

Note: This is a demo setup. In production, APK files should be properly signed and distributed through Google Play Store or other secure channels.
